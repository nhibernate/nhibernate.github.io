Open Source Project Ecosystem

NHibernate is itself an Open Source project that depends upon other open source projects to deliver its functionality. NHibernate also has a number of Open Source projects that include support for NHibernate or extend the functionality of NHibernate. A similar list of Commercial Products is here.
